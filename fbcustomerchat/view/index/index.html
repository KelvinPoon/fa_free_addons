快速配置说明

第一步：
在FA后台中填写你的Facebook Page ID。
第二步：
确保浏览器可正常访问www.facebook.com。
第三步：
访问www.facebook.com，并在公共主页的设置菜单中点击“Messenger平台”，将你当前的网站域名添加至白名单（请确保你的域名可通过https方式访问）。
刷新此页面，如果右下角出现了Messenger插件图标，表示已经大功告成。

常见问题

按快速配置说明操作后，仍然看不到右下角的Messenger插件图标？
1.请打开浏览器调试工具（F12），如果在console见到下方信息，表示当前网站域名仍然未添加至白名单。
Refused to display 'https://www.facebook.com/v3.2/plugins/customerchat.php?app_id=&attribution=setup_tool...' in a frame because an ancestor violates the following Content Security Policy directive: "frame-ancestors https://www.facebook.com".
2.如果你使用Firefox桌面浏览器（版本 63 及以上），请确认“内容拦截”功能是否屏蔽了插件（在搜索栏中点击灰色盾牌图标）。
